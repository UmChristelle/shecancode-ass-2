# Explore Kigali Website

## Project Description
This project is a simple multi-page website created to showcase places to visit, things to do, and places to eat around Kigali. It demonstrates the use of HTML for structure and CSS for styling, following web development best practices.

## HTML Pages Included
- index.html (Home page)
- kigali-city.html
- restaurants.html
- activities.html
- nightlife.html

## What This Project Demonstrates
- Proper HTML page structure and semantic elements
- Use of images, lists, tables, audio, video, and embedded content
- Internal and external navigation links
- CSS styling with separate CSS files for each page
- Organized project structure
- Use of GitHub for version control with multiple meaningful commits

## Technologies Used
- HTML5
- CSS3
